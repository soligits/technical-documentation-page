Introduction

Visual Studio Code is a source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git. Users can change the theme, keyboard shortcuts, preferences, and install extensions that add additional functionality. In the Stack Overflow 2021 Developer Survey, Visual Studio Code was ranked the most popular developer environment tool, with 70% of 82,000 respondents reporting that they use it.

Installation

Windows

For windows users, the installation proccess is really simple. All you need to do is to download the executable file that is compatilble with your processor from the VS Code download section. After that, just open the file, and proceed with the installation.

Linux

Snap

If your OS supports snap store, then open a terminal and run the following code: 

sudo snap install --classic code

Debian and Ubuntu based distros

For Ubuntu and other Debian based distros, you can also download the .deb package. You can also use wget for this: 

wget https://go.microsoft.com/fwlink/?LinkID=760868
After that, install the .deb file using apt: 

sudo apt install {file}.deb
You can update the package using the code bellow: 

sudo apt install apt-transport-https
sudo apt update
sudo apt install code # or code-insiders

RHEL, Fedora, and CentOS based distributions

Install the yum repository and key: 

sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc
sudo sh -c 'echo -e "[code]\nname=Visual Studio Code\nbaseurl=https://packages.microsoft.com/yumrepos/vscode\nenabled=1\ngpgcheck=1\ngpgkey=https://packages.microsoft.com/keys/microsoft.asc" > /etc/yum.repos.d/vscode.repo'
And then update: 

dnf check-update
sudo dnf install code
